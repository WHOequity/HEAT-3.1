Software

The toolkit was developed using R: A Language and Environment for Statistical Computing and shiny: Web Application Framework for R. Highcharts, Highmaps and Highstock wer used to create the interactive visuals and maps displayed in HEAT.

 

Additional R packages used to support the toolkit include:

  • broom: convert statistical analysis objects into tidy tibbles
  • data.table: extension of 'data.frame'
  • dplyr: a grammar of data manipulation
  • DT: a wrapper of the JavaScript library 'data.tables'
  • ggplot2: create elegant data visualisations using the grammar of graphics
  • grid: the grid graphics package
  • gridExtra: miscellaneous functions for 'grid' graphics
  • highcharter: a wrapper for the 'highcharts' library
  • htmltools: tools for HTML
  • knitr: a general-purpose package for dynamic report generation in R
  • purr: functional programming tools
  • RColorBrewer: ColorBrewer palettes
  • scales: scale functions for visualization
  • shinyBS: twitter bootstrap components for shiny
  • stringr: simple, consistent wrappers for common string operations
  • readxl: read excel files
  • tidyr: easily tidy data with 'spread()' and 'gather()' functions
  • viridis: default color maps from 'matplotlib'

The standalone version of the toolkit uses a portable version of R and the portable edition of the Google Chrome web browser.
